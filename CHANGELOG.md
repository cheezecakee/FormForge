# Changelog

All notable changes to Form Forge will be documented here.

## [1.0.0] - 2026-06-25

### Added
- Profile creation and editing with job application data including personal info,
  contact details, work history, education, skills, languages, cover letters, and FAQ answers
- Profile viewer in read-only mode
- Clear profile option
- Supported sites management — add and remove custom job sites
- Semi-automatic form filling on Greenhouse job applications — select a field
  then select a value from the panel to fill it
- Copy to clipboard fallback when no field is selected
- Help guide with usage instructions and screenshots
- Job site indicator in popup showing whether Forge is active on the current page
