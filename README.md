# Form Forge

A browser extension that helps you reuse your job application information across websites.

## Overview

Form Forge lets you create a profile once and reuse that information when applying for jobs.

Instead of repeatedly entering the same details, Forge stores your information locally and helps you quickly fill or copy application fields.

## Features

- Create a reusable job application profile
- Store:
  - Personal information
  - Work experience
  - Education
  - Projects
  - Skills
  - Frequently asked questions
  - Cover letters
- Quickly access saved information while applying
- Fill supported application fields automatically
- Copy information for fields that require manual input
- Manage multiple saved responses

## How It Works

1. Create your Forge profile
2. Visit a job application website
3. Open Forge when you need information
4. Select the content you want to use
5. Fill the field or copy the information

## Installation

### Chrome / Chromium browsers

1. Download the latest release
2. Open your browser extension page
3. Enable developer mode
4. Load the extension
5. Open Forge from your extension menu

## Privacy

Form Forge does not store your data on external servers.

All profile information is stored locally in your browser and remains on your device.

Form Forge does not require an account or login.
